# Logo Assets

This directory contains all logo assets for the LKS Multilingual Translator application.

## Available Logos

- `logo-hero.jpg` - Full logo with text for the main hero section
- `logo-header.jpg` - Simplified logo for the header/navigation
- `favicon.ico` - Browser tab icon (also in public/ root)

## Color Palette

- Navy Blue: `#1e3a8a` (primary color)
- Gold/Yellow: `#fbbf24` (accent color)
- White: `#ffffff` (text color)

## Usage

These logos are referenced in:
- `components/Header.tsx` (uses logo-simple.svg)
- `App.tsx` (uses logo.svg in hero section)
- `index.html` (uses favicon.ico)

## Customization

To customize these logos:
1. Replace the SVG files with your own designs
2. Ensure the dimensions match the current ones for proper display
3. Update colors to match your brand if needed
